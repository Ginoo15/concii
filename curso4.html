Landing Page para Cafetería

Un diseño moderno y minimalista para una marca local de café artesanal.

Este proyecto se enfocó en crear una página web atractiva para la cafetería “Aromas”. Se aplicaron principios de UX/UI y se usaron herramientas como Figma y HTML/CSS. El sitio incluye secciones de menú, historia y formulario de contacto con diseño responsivo.

Portfolio Personal

Un sitio web donde cada estudiante muestra sus proyectos y habilidades.

El objetivo de este trabajo fue que cada alumno construya un portfolio propio, integrando animaciones, navegación fluida y un diseño adaptable. Se trabajó con HTML5, CSS3 y principios de identidad visual.

Sitio Institucional Escolar

Desarrollo de una página informativa con calendario y noticias.

Este proyecto grupal propuso modernizar la web del colegio, incluyendo apartados como noticias, eventos, inscripciones y contacto. Se aplicaron buenas prácticas de accesibilidad y diseño adaptativo.
